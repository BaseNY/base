## Naming Convention

* Javascript: Camel case
```javascript
var variableName;
function functionName() {}
```

* Template names: Camel case
```html
<template name="templateName"></template>
```

* CSS selector: Hyphens
```css
.class-name { color: black; }
```

* File names: underscore
```
file_name.html
```
